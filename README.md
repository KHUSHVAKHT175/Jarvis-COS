# ?? Jarvis-COS Minimal
> Минималистская версия искусственного интеллекта Jarvis-COS, предназначенная для локального запуска, обучения и расширения.

> Операционная система нового типа с архитектурой I1/I2: логика, исполнение, самообучение, временные поля и машинный словарь.

---

## ?? Основные компоненты

| Компонент           | Назначение                                                     |
|---------------------|---------------------------------------------------------------|
| **I1**              | Логический/проектировочный интеллект (анализ, организация)    |
| **I2**              | Исполнительный интеллект (выполнение, коррекция)              |
| **Tom A**           | История машинно-человеческого понимания                       |
| **Tom 1**           | Машинный словарь глаголов                                     |
| **Модуль 7D**       | Восприятие и внимание                                         |
| **Модуль 11D**      | Переписчик времени, работа с ветвями истории                  |
| **CoDeveloper**     | Локальный генератор кода/исправлений                         |
| **GUI Monitor**     | Визуализация состояния, логов, решений                        |
